# 规则详解

本文档详细介绍 `@ai-partner-x/eslint-plugin-aiko-boot` 提供的所有规则。

## 基础规则

### no-arrow-methods

**禁止箭头函数作为类方法**

Java 不支持箭头函数作为类方法，必须使用标准的方法定义语法。

❌ **错误示例**：

```typescript
@RestController()
class UserController {
  // 箭头函数作为方法
  getUser = async (id: number) => {
    return this.userService.getUserById(id);
  }
}
```

✅ **正确示例**：

```typescript
@RestController()
class UserController {
  // 标准方法定义
  async getUser(id: number) {
    return this.userService.getUserById(id);
  }
}
```

**Java 转译结果**：

```java
@RestController
public class UserController {
    public User getUser(Integer id) {
        return this.userService.getUserById(id);
    }
}
```

---

### no-destructuring-in-methods

**禁止方法中使用解构**

Java 不支持解构赋值语法，需要使用显式的属性访问。

❌ **错误示例**：

```typescript
async createUser(dto: CreateUserDto) {
  const { username, email } = dto;  // 解构
  return this.userService.create(username, email);
}
```

✅ **正确示例**：

```typescript
async createUser(dto: CreateUserDto) {
  const username = dto.username;  // 显式属性访问
  const email = dto.email;
  return this.userService.create(username, email);
}
```

**Java 转译结果**：

```java
public User createUser(CreateUserDto dto) {
    String username = dto.getUsername();
    String email = dto.getEmail();
    return this.userService.create(username, email);
}
```

---

### no-object-spread

**禁止对象展开运算符**

Java 不支持 `...` 展开运算符，需要使用显式的属性赋值。

❌ **错误示例**：

```typescript
async updateUser(id: number, dto: UpdateUserDto) {
  const user = { ...dto, id };  // 对象展开
  return this.userRepository.save(user);
}
```

✅ **正确示例**：

```typescript
async updateUser(id: number, dto: UpdateUserDto) {
  const user = new User();
  user.id = id;
  user.username = dto.username;
  user.email = dto.email;
  return this.userRepository.save(user);
}
```

**Java 转译结果**：

```java
public User updateUser(Integer id, UpdateUserDto dto) {
    User user = new User();
    user.setId(id);
    user.setUsername(dto.getUsername());
    user.setEmail(dto.getEmail());
    return this.userRepository.save(user);
}
```

## 控制器规则

### static-route-paths

**强制路由路径为静态字符串**

路由路径必须是字符串字面量，不能使用变量或表达式。

❌ **错误示例**：

```typescript
const BASE_PATH = '/api/users';

@RestController({ path: BASE_PATH })  // 变量引用
class UserController {}

@GetMapping(`/users/${id}`)  // 模板字符串
async getUser() {}
```

✅ **正确示例**：

```typescript
@RestController({ path: '/api/users' })  // 字符串字面量
class UserController {
  @GetMapping('/:id')  // 字符串字面量
  async getUser() {}
}
```

---

### require-rest-controller

**要求使用 @RestController 装饰器**

包含路由映射装饰器（如 `@GetMapping`）的类必须有 `@RestController` 装饰器。

❌ **错误示例**：

```typescript
class UserController {  // 缺少 @RestController
  @GetMapping('/:id')
  async getUser(id: number) {}
}
```

✅ **正确示例**：

```typescript
@RestController({ path: '/api/users' })
class UserController {
  @GetMapping('/:id')
  async getUser(id: number) {}
}
```

## Java 兼容规则

### no-optional-chaining-in-methods

**禁止方法中使用可选链**

Java 不支持 `?.` 可选链运算符。

❌ **错误示例**：

```typescript
async getUsername(user: User | null) {
  return user?.username;  // 可选链
}
```

✅ **正确示例**：

```typescript
async getUsername(user: User | null) {
  if (user !== null) {
    return user.username;
  }
  return null;
}
```

---

### no-nullish-coalescing

**禁止空值合并运算符**

Java 不支持 `??` 空值合并运算符。

❌ **错误示例**：

```typescript
async getPage(params: SearchParams) {
  const page = params.page ?? 1;  // 空值合并
}
```

✅ **正确示例**：

```typescript
async getPage(params: SearchParams) {
  const page = params.page !== undefined ? params.page : 1;  // 三元运算符
}
```

---

### explicit-return-type

**强制显式返回类型**

方法必须有显式的返回类型声明。

❌ **错误示例**：

```typescript
async getUser(id: number) {  // 缺少返回类型
  return this.userMapper.selectById(id);
}
```

✅ **正确示例**：

```typescript
async getUser(id: number): Promise<User | null> {
  return this.userMapper.selectById(id);
}
```

---

### no-union-types

**禁止联合类型**

Java 不支持 TypeScript 的联合类型。

❌ **错误示例**：

```typescript
interface SearchParams {
  orderBy: 'asc' | 'desc';  // 联合类型
}
```

✅ **正确示例**：

```typescript
interface SearchParams {
  orderBy: string;  // 使用基础类型，注释说明可选值
  // 可选值: 'asc', 'desc'
}
```

---

### no-inline-object-types

**禁止内联对象类型**

方法参数和返回类型不能使用内联对象类型，必须定义为接口或类型别名。

❌ **错误示例**：

```typescript
async search(): Promise<{ data: User[]; total: number }> {  // 内联对象类型
  // ...
}
```

✅ **正确示例**：

```typescript
interface SearchResult {
  data: User[];
  total: number;
}

async search(): Promise<SearchResult> {
  // ...
}
```

## 规则速查表

| 规则 | 问题代码 | 修复方法 |
|------|----------|----------|
| no-arrow-methods | `getUser = () => {}` | `getUser() {}` |
| no-destructuring-in-methods | `const { a, b } = obj` | `const a = obj.a` |
| no-object-spread | `{ ...obj }` | 显式属性赋值 |
| static-route-paths | `@GetMapping(PATH)` | `@GetMapping('/path')` |
| require-rest-controller | 缺少装饰器 | 添加 `@RestController` |
| no-optional-chaining-in-methods | `obj?.prop` | `obj !== null ? obj.prop : null` |
| no-nullish-coalescing | `a ?? b` | `a !== undefined ? a : b` |
| explicit-return-type | `getUser()` | `getUser(): User` |
| no-union-types | `'a' \| 'b'` | `string` |
| no-inline-object-types | `{ a: string }` | 定义接口 |
